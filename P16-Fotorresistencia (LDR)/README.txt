By implementing an Arduino NANO development board and an LDR, the level of light to 
which the environment is exposed will be detected by increasing or decreasing the 
LDR and based on that, when there is low illumination, a spotlight will be turned 
on.

/* -------------------------------------------------------------------------------- */

Implementando una placa de desarrollo Arduino NANO y una LDR, se detectará el nivel 
de luz a la que está expuesto el ambiente aumentando o disminuyendo el LDR y en base 
a eso, Cuando exista una iluminación baja, se prendera un foco.
