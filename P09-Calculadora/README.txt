By implementing a 16x2 LCD display along with an I2C interface, an Arduino Nano 
development board, and a keypad, a system will be developed to perform basic 
calculator operations, providing user feedback on the LCD with the expected results.

/* -------------------------------------------------------------------------------- */

Implementando un Display LCD de 16x2 junto con una interfaz I2C, una placa de 
desarrollo Arduino nano, y un Keypad se desarrollará un sistema que funcione con las 
operaciones básicas de una calculadora, mostrando en el LCD una retroalimentación al 
usuario con los resultados esperados.
