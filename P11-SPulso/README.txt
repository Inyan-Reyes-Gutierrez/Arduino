Implementando un Sensor Infrarrojo de detección de ritmo cardiaco para dedo junto con 
una tarjeta de desarrollo de tipo Arduino, se censará el ritmo cardiaco y se 
desplegara en el monitor serial de una computadora.

/* -------------------------------------------------------------------------------- */

By implementing an Infrared Finger Heart Rate Detection Sensor together with an 
Arduino development board, the heart rate will be sensed and displayed on the serial 
monitor of a computer.
