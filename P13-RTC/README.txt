Implementing an Arduino NANO development board together with Real time Clock (RTC), 
we will display on a 16x2 LCD Display the time extracted from it, allowing the user 
to know the exact time and day.

/* -------------------------------------------------------------------------------- */

Implementando una placa de desarrollo Arduino NANO junto con Real time Clock o por su 
traducción Reloj de Tiempo Real (RTC), desplegaremos en un Display LCD de 16x2 la 
hora que se extrajo del mismo, permitiendo al usuario conocer la hora y el día 
exacto.
