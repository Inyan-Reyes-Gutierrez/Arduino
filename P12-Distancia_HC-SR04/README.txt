Implementing an Arduino NANO development board together with the HC-SR04 ultrasonic 
sensor will perform distance measurements, marking how many meters or centimeters 
away an object is within the range of the sensor, this will be displayed by means 
of a 16x2 LCD.

/* -------------------------------------------------------------------------------- */

Implementando una placa de desarrollo Arduino NANO junto con el sensor ultrasónico 
HC-SR04 se realizará mediciones de distancia, marcando a cuantos metros o centímetros 
esta un objeto dentro del rango del sensor, esta se vera desplegada por medio de un 
LCD 16x2.
