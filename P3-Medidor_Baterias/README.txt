By using a 16x2 LCD display and an Arduino Nano development board, a system should 
be implemented with the capability to measure AA and AAA batteries, providing 
information about them, and based on certain voltage parameters, give a verdict on 
their condition.

/* -------------------------------------------------------------------------------- */

Implementando un Display LCD de 16x2 y una placa de desarrollo Arduino nano, se 
deberá realizar un sistema con la capacidad de medir baterías AA y AAA, dotando 
información de estas y en base a ciertos parámetros de voltaje, dar un veredicto 
sobre su estado.
